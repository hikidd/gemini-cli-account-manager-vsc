# 更新日志 (Change Log)

这里记录了 "Gemini CLI Account Manager" 插件的所有重要更新。

## [1.2.13] - 2026-03-09

### 优化与改进 (Changed)
- **高级账号头像圆环优化**：进一步压缩了彩色圆环的边距，使其更加紧凑，视觉还原度更接近 Gemini Advanced 原生效果。
- **模型列表修正**：将 Gemini 3 Flash 的显示名称及 ID 从错误扩展的 3.1 修正为官方版本 3。

## [1.2.12] - 2026-03-09

### 优化与改进 (Changed)
- **视觉还原修复**：修正了高级账号头像圆环的几何分布。将黄色区域精确对齐至 9 点钟（正左侧）位置，并严格按照官方比例大幅缩小了黄色占比，确保 UI 视觉效果与官方应用高度一致。

## [1.2.11] - 2026-03-09

### 优化与改进 (Changed)
- **像素级 UI 还原**：深度参考 Google 品牌几何规范，重新定义了头像彩色圆环的颜色占比与分布。大幅增加了蓝色的主视觉占比，并将黄色精简为左侧微小片段，同时引入了平滑渐变效果，使 UI 更加精致专业。

## [1.2.10] - 2026-03-09

### 优化与改进 (Changed)
- **UI 细节优化**：精确调整了高级账号头像彩色圆环的颜色比例。根据 Google 官方视觉规范，大幅增加了蓝色和红色的占比，并缩小了黄色的显示区域，使整体视觉效果更加专业、自然。

## [1.2.9] - 2026-03-09

### 修复 (Fixed)
- **UI 修复**：修复了 1.2.8 版本中由于样式冲突导致的高级账号头像彩色圆环不显示的问题。重构了头像组件结构，确保彩色边框在各种主题下均能完美呈现。

## [1.2.8] - 2026-03-09

### 新增功能 (Added)
- **自动登录保活**：实现了 Token 自动刷新机制。当账号的登录凭证即将过期时，插件会自动利用 Refresh Token 获取新的凭证，确保持续可用，无需频繁重新登录。
- **自动同步**：刷新后的凭证会自动同步至 Gemini CLI 配置文件，确保终端操作不受中断。

## [1.2.7] - 2026-03-09

### 优化与改进 (Changed)
- **视觉体验升级**：根据 Google 品牌风格优化了高级账号（PRO/ULTRA）的头像彩色圆框样式，增加了间隙效果和硬切分色彩分布。

## [1.2.6] - 2026-03-09

### 优化与改进 (Changed)
- **模型支持更新**：更新了 Gemini 模型的选择列表，支持最新的 Gemini 3.1 Pro 和 Gemini 3.1 Flash 模型。
- **界面优化**：更新了配额显示逻辑，适配 3.1 系列模型。
- **发布配置优化**：优化了打包排除规则，减小插件体积。

## [1.2.5] - 2026-03-09
- 内部发布版本。

## [1.2.3] - 2026-02-11

### 新增功能 (Added)
- **历史会话管理**：现在你可以直接查看和管理 Gemini CLI 的历史会话记录了。
  - 新增了 "Sessions" (历史会话) 标签页，支持在账号列表和会话列表间切换。
  - 支持查看会话预览、消息条数以及最后更新时间。
  - **加载会话**：一键加载之前的对话上下文，快速恢复工作状态。
  - **删除会话**：直接在 VS Code 中清理不再需要的历史记录。
- 新增了工作区自动检测功能，确保打开文件夹时能自动加载会话数据。

### 优化与改进 (Changed)
- 优化了账号卡片和会话列表的 UI 布局，界面更加清爽。
- 修复了在部分文件系统下路径大小写敏感导致无法找到历史记录的问题。

## [1.0.0] - 2025-01-01
- 初始版本发布
