# 更新日志 (Change Log)

这里记录了 "Gemini CLI Account Manager" 插件的所有重要更新。

## [1.2.1] - 2026-02-11

### 新增功能 (Added)
- **历史会话管理**：现在你可以直接查看和管理 Gemini CLI 的历史会话记录了。
  - 新增了 "Sessions" (历史会话) 标签页，支持在账号列表和会话列表间切换。
  - 支持查看会话预览、消息条数以及最后更新时间。
  - **加载会话**：一键加载之前的对话上下文，快速恢复工作状态。
  - **删除会话**：直接在 VS Code 中清理不再需要的历史记录。
- 新增了工作区自动检测功能，确保打开文件夹时能自动加载会话数据。

### 优化与改进 (Changed)
- 优化了账号卡片和会话列表的 UI 布局，界面更加清爽。
- 修复了在部分文件系统下路径大小写敏感导致无法找到历史记录的问题。
