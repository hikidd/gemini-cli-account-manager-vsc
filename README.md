# KIRO 萝卜助手

KIRO 萝卜助手扩展，用于管理授权、代理设置和查看账号状态。

## 功能特性

✅ **授权验证** - 输入授权码快速验证 VIP 身份  
✅ **代理控制** - 一键开启/关闭代理功能  
✅ **状态查看** - 实时查看剩余次数、使用情况和有效期  
✅ **账号管理** - 支持快速切换账号和重新登录  
✅ **数据持久化** - 自动保存设置，重启 VS Code 后无需重新配置  

## 使用方法

### 1. 打开管理面板

- 按 `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`) 打开命令面板
- 输入 "KIRO CARROT: 打开管理面板"
- 或者在命令面板中搜索 "KIRO CARROT"

### 2. 验证授权码

1. 在授权验证区域输入您的授权码
2. 点击"验证"按钮
3. 验证成功后会显示授权状态信息

### 3 其他功能

- **一键切换新账号** - 清除当前授权信息，切换到新账号
- **重新登录** - 保留授权码，重新获取最新状态
- **使用教程** - 打开详细使用教程
- **联系方式** - 点击 QQ 号码可复制到剪贴板
